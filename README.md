# OfficeCAD

CAD projects for a home office environment, primarily using FreeCAD.

## Projects

### [Prusa Frame 6U Rack Mount Brackets](prusa-rack-conversion/)

3D-printable L-brackets to mount standard 6U rack rails to a Prusa MK3S/MK4S 3D printer frame. Uses existing z-axis motor screw locations for non-destructive mounting.

**Status:** Complete

## Software

- **CAD Software:** [FreeCAD](https://www.freecadweb.org/) (0.21+)
- **3D Printer Slicer:** [PrusaSlicer](https://www.prusa3d.com/page/prusaslicer_424/)

## Hardware

- **3D Printer:** Prusa Core One
  - Build plate: 250mm (X) x 210mm (Y)
- **Recycled Parts:** MK3S/MK4S parts from Core One upgrade
